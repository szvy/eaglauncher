Indev

Indev was the third phase of Minecraft's development cycle and the oldest version with Minecraft's Survival mode, which succeeded Survival Test in Classic Mode, and was the second Minecraft development stage to have some of its versions released to the public.

Beta 1.3

Beta 1.3 is a major update to Java Edition released on August 1, 2012, which added trading, adventure mode, new generated structures (jungle and desert pyramids) and new blocks/items including emeralds, book and quills, written books, and tripwire.

Beta 1.7.3

Beta 1.7.3 is a version of Java Edition Beta released on July 8, 2011, to fix some of the bugs and crashes in Beta 1.7.2

1.2.5

1.2.5 is a minor update to Minecraft from April 4, 2012, which improves functionality of the pick block key, and fixes bugs.

1.5.2

1.5.2 is a minor update for Java Edition, released on May 2, 2013, to fix some of the bugs and crashes in 1.5 and 1.5.1. This is the last official Minecraft version that will run on PowerPC Mac computers and Java 5, and the last version to be playable with no download required.

1.8

1.8, the first release of the Bountiful Update, is a major update to Minecraft. It added and changed many aspects of Minecraft. Among many things, the update made it easier for map makers to create adventure maps, while using the newly added Spectator mode. It added many blocks, mobs, and a structure for Survival play. Many commands were added or extended, the world boundary became a customizable component, many Survival mechanics were changed, and the update gave way for many more world customization options.
